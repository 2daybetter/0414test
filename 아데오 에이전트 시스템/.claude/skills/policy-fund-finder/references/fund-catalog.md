## 1. 창업지원 자금

### 창업중소기업 세제 혜택 (국세청)

| 항목 | 내용 |
|------|------|
| 대상 | 창업 5년 이내 중소기업 |
| 지원 내용 | 법인세·소득세 50% 감면 (3~5년) |
| 지원 기관 | 국세청 |
| 신청 방법 | 세무서 신청 또는 홈택스 |

### 창업기업 지원사업 (중소벤처기업부)

| 항목 | 내용 |
|------|------|
| 대상 | 업력 7년 이내 창업기업 |
| 지원 내용 | 사업화 자금 (최대 1억) |
| 지원 기관 | 창업진흥원 |
| 신청 기간 | 연 1~2회 공모 (K-스타트업 공고) |
