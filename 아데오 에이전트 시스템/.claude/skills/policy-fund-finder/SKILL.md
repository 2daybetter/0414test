# policy-fund-finder

## 개요

고객사 기업 정보 기반으로 적합 정책자금을 매칭하고 신청 전략 보고서(PR-02)를 생성한다.  
반드시 `/templates/policy-fund-template.md` 를 참조하여 작성한다.

---

## 실행 절차

### Step 1: 고객사 적격성 정보 수집

| 필수 항목 | 확인 방법 |
|---------|---------|
| 기업명 | 입력 |
| 기업 규모 | 중소기업/중견/대기업 (상시 근로자 수, 매출액) |
| 업종 (KSIC) | 한국표준산업분류 코드 |
| 설립 연도 | 창업 기간 산정 기준 |
| 소재지 | 광역시/도 기준 |
| 기술 인증 | 이노비즈, 벤처, 기술혁신형 등 |

정보 미제공 시 **[확인 요청]** 출력 후 중단.

### Step 2: 정책자금 조회

세부 자금 목록 및 기준은 `references/fund-catalog.md` 참조.

다음 유형 순으로 적합 자금을 탐색:

1. **창업지원**: 설립 7년 이내 기업 대상
2. **R&D 지원**: 기술 개발/혁신 과제
3. **디지털 전환**: IT 시스템 구축 관련
4. **수출/글로벌화**: 해외 진출 계획 있는 경우
5. **지역 특화**: 소재지 기반 지자체 지원사업

### Step 3: 매칭 결과 작성

| 항목 | 내용 |
|------|------|
| 적합 자금명 | 공식 사업명 |
| 지원 기관 | 중소벤처기업부, 창진원, 지자체 등 |
| 지원 내용 | 보조금/융자/컨설팅 등 유형 + 금액 |
| 신청 요건 | 기업 규모, 업종, 기간 제한 등 |
| 신청 기간 | 공고 일정 (모집 공고 기준) |
| 적합도 | 상/중/하 + 근거 |
| 신청 우선순위 | 1~3순위 추천 |

### Step 4: 출력 및 저장

- **출력 방법**: 내용 작성 후 `mcp__claude_ai_Google_Drive__create_file`로 업로드 → 반환된 URL을 Google Drive 아데오 프로젝트/{기회명}/.status/.status 파일의 `outputs.policy-fund`에 기록
- **성공 기준**: 적합 자금 1개 이상 + 신청 요건 명시 + 우선순위 제시

## 실패 처리

| 실패 유형 | 처리 |
|---------|------|
| 고객사 정보 불충분 | [확인 요청] 출력 후 중단 |
| 매칭 자금 0건 | 업종/규모 재확인 후 재조회 1회 → 결과 보고 |
| 공고 마감된 자금만 존재 | 차기 공고 예정 정보 포함 후 보고 |
