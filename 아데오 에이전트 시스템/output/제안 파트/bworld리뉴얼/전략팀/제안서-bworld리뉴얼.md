## 1. 제안 개요

### 1.1 제안 배경

bworld.co.kr은 현재 Java/.do 기반의 레거시 기술 스택으로 운영되고 있습니다. 구형 서버사이드 렌더링 구조로 인해 모바일 반응형 대응이 불완전하며, 페이지 로딩 속도와 사용자 경험(UX) 측면에서 경쟁사 대비 현저한 열위를 보이고 있습니다.

국내 모바일 웹 트래픽 비중이 전체의 70%를 상회하는 현시점에서, 반응형 미지원과 접근성 미흡은 직접적인 전환율 저하 및 이탈률 증가로 이어집니다. 또한 JavaScript 의존도가 높아 검색엔진 크롤링(SEO)이 어렵고, 신규 콘텐츠 등록·수정 등 운영 효율성도 낮은 상태입니다.

아데오는 이번 리뉴얼을 통해 최신 프론트엔드 기술 기반의 반응형 웹사이트로 전면 개편하고, 사용자 중심의 IA 재설계 및 운영 효율화를 달성할 것을 제안드립니다.

### 1.2 제안 목적

| 목적 | 내용 |
|------|------|
| 단기 목적 | 반응형 웹 구축 완료 및 페이지 속도 3초 이내 달성 |
| 중기 목적 | 모바일 전환율 40% 향상, 이탈률 20% 감소 |
| 장기 목적 | 지속 확장 가능한 CMS 기반 자체 운영 체계 확립 |

### 1.3 제안 범위

| 구분 | 범위 | 비고 |
|------|------|------|
| 포함 | FO 전면 리뉴얼 (반응형 웹), BO 관리자 시스템, IA 재설계, SEO 최적화 | |
| 포함 | 기존 콘텐츠 마이그레이션, 기술 스택 현대화 | |
| 제외 | 모바일 앱 개발 | 추후 2단계 예정 |
| 제외 | ERP·CRM 시스템 연동 | 요건 확정 후 협의 |

| 헤드리스 CMS 도입 | 비개발자도 콘텐츠 직접 관리 가능한 BO 구성 | 콘텐츠 업데이트 시간 70% 단축 |

| 레거시 마이그레이션 전문성 | Java/.do → Next.js 전환 경험 다수 보유, 기존 데이터 손실 없는 마이그레이션 |
| 운영 자립화 패키지 | 오픈 후 3개월 운영 교육 + 운영 매뉴얼(Figma) 포함 — 인하우스 운영 즉시 가능 |

- **운영 자립화**: 비개발자도 콘텐츠 직접 관리 → 운영 비용 절감
- **보안 강화**: 최신 보안 표준 적용으로 개인정보보호법 완전 준수
- **확장성 확보**: 컴포넌트 기반 설계로 향후 기능 추가 용이

### 9.3 아데오를 선택해야 하는 이유

> 1. **레거시 전환 전문성** — Java/.do에서 Next.js로의 기술 전환 경험 다수 보유  
> 2. **원스톱 구축** — 기획·디자인·개발·운영까지 단일 팀이 책임 완수  
> 3. **투명한 프로세스** — 단계별 산출물(IA, 화면설계서 등) 고객 승인 후 진행  
> 4. **오픈 후 케어** — 3개월 하자보수 + 운영자 교육 패키지 포함  
> 5. **합리적 금액** — 동급 품질 대비 경쟁력 있는 제안 금액
