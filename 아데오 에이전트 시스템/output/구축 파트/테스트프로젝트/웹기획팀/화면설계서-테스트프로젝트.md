## [FO_001] 메인 페이지

**화면 헤더**

| 항목 | 내용 |
|------|------|
| 화면ID | FO_001 |
| 화면명 | 메인 페이지 |
| URL 경로 | / |
| 버전 | v1.0 |
| 연관 화면ID | - |
| 작성일 | 2026-04-16 |

**ASCII 레이아웃**

```
+--[GNB: 로고 | 회사소개 | 사업영역 | 뉴스 | 고객지원 | ]--+
|                                                           |
|  [히어로 배너: 메인 이미지 + 슬로건]                        |
|  (더 알아보기)                                            |
|                                                           |
|  [섹션1: 사업영역 요약]                                    |
|  [카드] [카드] [카드]                                     |
|                                                           |
|  [섹션2: 최신 뉴스]                                       |
|  [뉴스카드] [뉴스카드] [뉴스카드]                          |
|  (더보기)                                                 |
|                                                           |
+--[Footer: 회사정보 | 연락처 | SNS]----------------------+
```

**기능 설명**

| 번호 | 구성요소 | 유형 | 기능 설명 | 비고 |
|------|---------|------|---------|------|
| 1 | GNB | 내비게이션 | 메뉴 클릭 시 해당 페이지 이동 | sticky |
| 2 | 히어로 배너 | 이미지+텍스트 | 자동 슬라이드 (3초 간격) | BO 관리 |
| 3 | 더 알아보기 버튼 | 버튼 | 클릭 시 회사소개 페이지 이동 | |
| 4 | 사업영역 카드 | 리스트 | 카드 클릭 시 해당 서비스 페이지 이동 | |
| 5 | 최신 뉴스 목록 | 리스트 | 클릭 시 공지사항 상세 FO_006 이동 | DB 연동 |
| 6 | 더보기 버튼 | 버튼 | 클릭 시 공지사항 목록 FO_005 이동 | |

**화면 조건**

| 조건 구분 | 내용 |
|---------|------|
| 로그인 전 | 기본 상태 — 모든 콘텐츠 표시 |
| 로그인 후 | 동일 (회원 기능 없음) |
| PC (1280px+) | 3컬럼 카드 레이아웃 |
| Mobile (375px~) | 1컬럼 카드, GNB 햄버거 메뉴 |
| 권한별 분기 | 없음 |

**ASCII 레이아웃**

```
+--[GNB]--------------------------------------------------+
|                                                          |
|  [브레드크럼: 홈 > 뉴스 > 공지사항]                       |
|                                                          |
|  [페이지 타이틀: 공지사항]                                 |
|  [검색: ___________________] (검색)                      |
|                                                          |
|  총 {N}건                                               |
|  +-------+------------------+----------+------+         |
|  | 번호  | 제목             | 날짜     | 조회 |         |
|  +-------+------------------+----------+------+         |
|  | {N}   | {제목}           | {날짜}   | {N}  |         |
|  | ...   | ...              | ...      | ...  |         |
|  +-------+------------------+----------+------+         |
|                                                          |
|  [페이지네이션: < 1 2 3 ... >]                           |
|                                                          |
+--[Footer]-----------------------------------------------+
```

**기능 설명**

| 번호 | 구성요소 | 유형 | 기능 설명 | 비고 |
|------|---------|------|---------|------|
| 1 | 브레드크럼 | 내비게이션 | 상위 메뉴 클릭 시 해당 페이지 이동 | |
| 2 | 검색창 | 폼 요소 | 제목 기준 키워드 검색, Enter 또는 버튼 클릭 | |
| 3 | 목록 행 | 리스트 | 클릭 시 FO_006 공지사항 상세 레이어 팝업 표시 | DB 연동 |
| 4 | 페이지네이션 | 내비게이션 | 페이지 이동, 페이지당 10건 | |

**화면 조건**

| 조건 구분 | 내용 |
|---------|------|
| 로그인 전 | 목록 조회 가능, 상세 보기 가능 |
| 로그인 후 | 동일 |
| PC (1280px+) | 테이블 레이아웃 |
| Mobile (375px~) | 카드형 목록으로 전환 |
| 권한별 분기 | 없음 |
