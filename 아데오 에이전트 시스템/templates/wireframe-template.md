## 공통 컴포넌트 정의

| 컴포넌트 | 설명 | 적용 위치 |
|----------|------|-----------|
| GNB | 상단 글로벌 네비게이션 | 전체 FO 페이지 |
| Footer | 하단 정보 영역 | 전체 FO 페이지 |
| 사이드바 | 관리자 좌측 메뉴 | 전체 BO 페이지 |
| 페이지네이션 | 목록 하단 페이지 이동 | 목록 페이지 |
| 알림 Toast | 처리 결과 알림 (상단 우측) | 전체 |
| 확인 Modal | 삭제/처리 확인 팝업 | 전체 |
| 날짜 선택기 | 날짜/기간 입력 | 검색, 등록 폼 |
| 파일 업로드 | 이미지/첨부파일 | 등록/수정 폼 |
