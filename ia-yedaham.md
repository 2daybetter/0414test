# 더케이예다함 정보구조(IA) 문서

> 작성일: 2026-04-20
> 대상: yedaham.co.kr (홈페이지) + 운영자 시스템(CMS) + 모바일 앱(React Native)

---

## 1. 전체 서비스 맵

```
더케이예다함 디지털 서비스
├── 홈페이지 (yedaham.co.kr)
│   ├── 공개 영역 (비로그인)
│   └── 회원 영역 (로그인 후)
├── 운영자 시스템 (CMS / 관리자)
└── 모바일 앱 (React Native)
```

## 2. 홈페이지 IA

### 2.1 글로벌 네비게이션

```
[헤더]
├── 로고 (홈 링크)
├── GNB 메뉴
│   ├── MY
│   ├── 상품안내
│   ├── 간편가입
│   ├── 혜택서비스
│   ├── 회사소개
│   └── 법인소개
├── 로그인 / 회원가입 CTA
└── 긴급 연락처 고정 표시 (전화 CTA)

[하단 고정 바 — 모바일]
├── 전화걸기
└── 상담신청
```

### 2.2 페이지 계층 구조

```
홈 (/)
│
├── MY (/my)
│   ├── 나의 정보 (/my/profile)
│   ├── 가입 상품 (/my/products)
│   ├── 혜택 현황 (/my/benefits)
│   └── 서비스 이용 내역 (/my/history)
│
├── 상품안내 (/products)
│   ├── 윤슬4 (/products/yunseul4)
│   │   ├── 상품 소개
│   │   ├── 보장 내역
│   │   ├── 가입 조건
│   │   └── 가입하기 CTA
│   ├── 윤슬6 (/products/yunseul6)
│   │   ├── 상품 소개
│   │   ├── 보장 내역
│   │   ├── 가입 조건
│   │   └── 가입하기 CTA
│   └── 노블 (/products/noble)
│       ├── 상품 소개
│       ├── 보장 내역
│       ├── 가입 조건
│       └── 가입하기 CTA
│
├── 간편가입 (/join)
│   ├── 상품 선택
│   ├── 본인인증 (NICE / 라온시큐어)
│   ├── 정보 입력
│   ├── 약관 동의
│   └── 가입 완료
│
├── 혜택서비스 (/benefits)
│   ├── 혜택 목록
│   ├── 제휴사 혜택
│   └── 혜택 상세
│
├── 회사소개 (/about) [일반 법인]
│   ├── 회사 개요
│   ├── 경영 철학 / 비전
│   ├── 연혁
│   ├── 조직도
│   ├── 수상 / 인증
│   └── 오시는 길
│
└── 법인소개 (/corporate) [기업 법인]
    ├── 법인 서비스 소개
    ├── 가입 절차
    ├── 문의하기
    └── 담당자 연락처
```

### 2.3 공통 페이지

```
공통 (/common)
├── 공지사항 (/notice)
│   ├── 목록
│   └── 상세
├── FAQ (/faq)
├── 개인정보처리방침 (/privacy)
├── 이용약관 (/terms)
├── 사이트맵 (/sitemap)
└── 오류 페이지 (404, 500)
```

### 2.4 홈(메인) 페이지 섹션 구조

```
홈 (/)
├── Hero 섹션
│   ├── 메인 카피 (안심 메시지)
│   ├── 긴급 접수 버튼 (Primary CTA)
│   └── 상담 신청 버튼 (Secondary CTA)
├── 상품 요약 섹션
│   ├── 윤슬4 카드
│   ├── 윤슬6 카드
│   └── 노블 카드
├── 신뢰 지표 섹션
│   ├── 업력
│   ├── 가입 회원 수
│   └── 서비스 수행 건수
├── 혜택 미리보기 섹션
├── 공지사항 최신 목록
└── 푸터
    ├── 회사 정보
    ├── 고객센터 번호
    ├── 관련 링크
    └── 법적 고지 (개인정보처리방침, 이용약관)
```

## 3. 운영자 시스템(CMS) IA

### 3.1 관리자 GNB

```
운영자 시스템 (관리자 전용)
├── 홈페이지관리
├── 회사사이트관리
├── 법인사이트관리
├── 제휴사관리
├── 통계관리
├── 이력관리
└── 시스템관리
```

### 3.2 모듈별 상세 구조

```
홈페이지관리
├── 메인 배너 관리
│   ├── 배너 목록
│   ├── 배너 등록/수정/삭제
│   └── 노출 순서 관리
├── 공지사항 관리
│   ├── 목록
│   └── 등록/수정/삭제
├── 팝업 관리
│   ├── 팝업 목록
│   └── 등록/수정/삭제 (기간·위치 설정)
├── FAQ 관리
└── 메뉴 구조 관리 (GNB 항목 노출 여부)

회사사이트관리
├── 회사 소개 콘텐츠 관리
├── 연혁 관리
├── 수상·인증 관리
└── 오시는 길 정보 관리 (지점별)

법인사이트관리
├── 법인 서비스 소개 콘텐츠 관리
├── 가입 절차 안내 관리
└── 담당자 정보 관리

제휴사관리
├── 제휴사 목록
├── 제휴사 등록/수정/삭제
└── 제휴 혜택 콘텐츠 관리

통계관리
├── 방문자 통계 (GA4 연동)
│   ├── 일별/주별/월별 방문 수
│   ├── 페이지별 방문 수
│   └── 채널별 유입 현황
├── 상품별 조회 통계
│   ├── 윤슬4 / 윤슬6 / 노블 페이지 방문
│   └── 가입하기 CTA 클릭 수
└── 광고 성과 현황 (채널별)

이력관리
├── 콘텐츠 변경 이력
│   ├── 변경 일시
│   ├── 변경 항목
│   ├── 변경 전/후 내용
│   └── 처리자
├── 배포 이력
└── 장애/결함 처리 이력

시스템관리
├── 관리자 계정 관리
│   ├── 계정 목록
│   ├── 계정 등록/수정/비활성화
│   └── 권한 설정 (모듈별 읽기/쓰기)
├── 공통 코드 관리
├── 메뉴 권한 설정
└── 시스템 설정 (사이트 기본 정보, 이메일 설정)
```

## 5. 콘텐츠 유형 정의

| 콘텐츠 유형 | 수정 주체 | 수정 방식 | 수정 빈도 |
|-----------|---------|---------|---------|
| 메인 배너 | 운영자 | CMS 홈페이지관리 | 수시 (프로모션) |
| 상품 정보 (윤슬4/6/노블) | 운영자 + 개발자 | CMS + 코드 수정 | 월 1~2회 |
| 혜택 서비스 정보 | 운영자 | CMS 홈페이지관리 | 수시 |
| 제휴사 정보 | 운영자 | CMS 제휴사관리 | 수시 |
| 공지사항 | 운영자 | CMS 홈페이지관리 | 수시 |
| 회사 소개 텍스트 | 운영자 | CMS 회사사이트관리 | 분기 1회 이하 |
| 법인 소개 텍스트 | 운영자 | CMS 법인사이트관리 | 분기 1회 이하 |
| 팝업 | 운영자 | CMS 홈페이지관리 | 수시 (이벤트) |
| 개인정보처리방침 / 약관 | 법무 확인 후 운영자 | CMS | 법적 변경 시 |

## 6. URL 네이밍 규칙

| 원칙 | 예시 |
|------|------|
| 소문자 영문 + 하이픈 | `/products/yunseul4` ✅ `/products/YunSeul4` ❌ |
| 계층 구조 반영 | `/my/products` (마이 > 가입 상품) |
| 한글 URL 금지 | `/products/yunseul4` ✅ `/products/윤슬4` ❌ |
| 동사형 경로 지양 | `/join` ✅ `/do-join` ❌ |
| ID 포함 시 끝에 배치 | `/notice/:id` |
